To connect to Anaplan data:

1. Select **Anaplan Connector** from the **Power Query - Choose data source** page.

2. In the **Anaplan Connector** dialog, enter the **API URL** and **Authentication URL** for your Anaplan instance. The default API URL is `https://api.anaplan.com` and the default authentication URL is `https://auth.anaplan.com`.

3. Select the **Basic** authentication kind, and enter your Anaplan username and password. Or select **Organizational account** if your organization uses SSO authentication with Anaplan.

4. Once you successfully sign in, select **Next**.

5. In the **Choose data** page, select the export actions you want to run, and then select **Transform data** to transform the data in Power Query Editor.
