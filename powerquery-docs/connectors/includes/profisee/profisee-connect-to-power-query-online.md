To connect to Profisee data:

1. Select **Profisee** from the **Power Query - Choose data source** page.

2. In the **Profisee** dialog, enter the **URL** for your Profisee instance.

3. Select **Sign in** to sign in with your organizational account.

4. Once you successfully sign in, select **Next**.

5. In the **Choose data** page, select the entities you want to load, and then select **Transform data** to transform the data in Power Query Editor.
