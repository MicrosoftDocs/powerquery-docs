If you're connecting to an Access database from Power Query Online, the system that contains the on-premises data gateway must have the 64-bit version of the Access Database Engine OLEDB provider installed.

If you're loading an Access database to Power BI Desktop, the versions of the Access Database Engine OLEDB provider and Power BI Desktop on that machine must match (that is, either 32-bit or 64-bit). For more information, go to [Import Access database to Power BI Desktop](/power-query/connectors/access-database#import-access-database-to-power-bi-desktop).
